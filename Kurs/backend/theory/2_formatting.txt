Урок 2. Форматирование текста

Форматирование делает текст более структурированным, удобным для чтения и презентабельным.

1. Шрифты
   - Выбор шрифта: Arial, Times New Roman, Calibri и др.
   - Размер: обычно 11–14 для основного текста, 16–24 для заголовков.
   - Цвет: можно использовать стандартные цвета или выбрать пользовательский.

2. Стили текста
   - Жирный (Ctrl + B)
   - Курсив (Ctrl + I)
   - Подчёркивание (Ctrl + U)
   - Зачёркивание, верхний и нижний индекс (например: X², H₂O)

3. Выравнивание текста
   - По левому краю, по центру, по правому краю, по ширине (Ctrl + L / E / R / J)
   - Настройка отступов и табуляции

4. Межстрочный интервал
   - Стандарт: 1.15 или 1.5
   - Настройка через вкладку «Абзац»

5. Списки
   - Маркированные (•, –)
   - Нумерованные (1., 2., 3.)
   - Многоуровневые списки

6. Использование стилей
   - Применение предустановленных стилей (Заголовок 1, Заголовок 2, Основной текст)
   - Можно создавать свои стили оформления

Советы:
- Выделите текст и нажмите правую кнопку мыши для быстрого доступа к форматированию.
- Используйте «Формат по образцу» (кисточка) для копирования форматирования.
